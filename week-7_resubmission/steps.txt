The propose of this project is to build VPC in GCP using Terraform

Tools
-Google Cloud Platform
-Terraform
-Github
-VS Code

Steps
1. Create a local repository 
2. Create the week7 directory & the infra directory inside it
3. Run gcloud init
4. Add the GCP provider
5. Add the VPC name
6. Add the desired Output information 
7. Ran Terraform init
8. Terraform validate
9. Terraform plan
10. Terraform apply -auto-approve
11. Screenshot my terminal results for plan and apply
12. Screenshot my VPC
13. Git Push
